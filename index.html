프로젝트 소개
음반 전문 C2C 중고거래 플랫폼
CD·LP·카세트·DVD 수집가를 위한 버티컬 마켓플레이스. 별도 연락 없이 플랫폼 안에서 바로 소통할 수 있는 실시간 채팅과 음반 특화 필터링으로 오프라인 레코드샵 디깅 경험을 온라인으로.
문제 인식
소통 불편
기존 플랫폼의 소통 장벽
알라딘·예스24는 실물 사진 요청 시 판매자에게 직접 연락하거나 이메일 Q&A를 이용해야 함. 즉각적인 소통이 어렵고 거래 접근성이 떨어짐.
전문성 부족
종합 C2C의 한계
번개장터·당근마켓은 소통은 편하지만 포맷·세부 장르 등 음반 특화 필터가 없어 원하는 매물 탐색 시 무관한 게시글 피로감이 큼.
해결 방안
💬
닉네임 기반 1:1 실시간 채팅
별도 연락처 교환이나 외부 메신저 없이 플랫폼 안에서 바로 소통 가능. 실물 사진 요청, 가격 협의, 거래 진행까지 한 곳에서 즉시 처리할 수 있는 환경 제공.
🔍
음반 탐색에 최적화된 버티컬 검색
포맷(LP·CD·카세트·DVD)과 대/소분류 장르를 조건으로 하는 다중 필터링. 오프라인 레코드샵에서 디깅하는 듯한 전문적 탐색 경험 제공.
🛡️
신뢰 기반 거래 환경 조성
본인인증 완료 사용자만 판매 가능하도록 설계하고, 거래 후 상점 리뷰 시스템을 도입해 플랫폼 직접 검수 없이도 사용자 간 자발적 신뢰가 쌓이는 환경 제공
